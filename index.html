Tentang Kami

SYSCO FIRE

PT. Somawi Surya Semesta adalah perusahaan terdepan yang memproduksi alat pemadam api (fire extinguisher) dengan merek Sysco Fire.

Alat pemadam api Sysco Fire telah menjadi salah satu pilihan terbaik untuk melindungi kita semua dari bahaya kebakaran, dengan kualitas produk prima dan layanan purna jual spare part dan isi ulang yang terjamin. Perusahaan kami menyediakan sistem pemadam api terintegrasi :

Smoke Detector
Gas Detector
Fire Alarm
Hydrant
Tentang Kami

Terpercaya

Alat Pemadam Api merek SYSCO FIRE telah dipakai di gedung-gedung perkantoran, perhotelan, BUMN, perumahan, real estate & SPBU / SPBE/ Transportir Pertamina di seluruh Indonesia.

Inovatif

Sysco Fire adalah merek produk alat pemadam api yang inovatif. Sysco Fire alat pemadam api yang tersedia dalam berbagai ukuran APAR portable maupun Troly, baik dengan sistem Store Pressure / Cartridge Operated dengan media ABC Dry Chemical Powder, Foam dan Carbon Dioxide (CO2).

Berkualitas

Alat Pemadam Api Merek SYSCO FIRE telah menjadi salah satu pilihan terbaik untuk melindungi kita semua dari bahaya kebakaran, dengan kualitas produk yang prima dan layanan purna jual spare part dan isi ulang yang terjamin
